SPA - Single Page Application
.NET CORE 10 CLI, GRAPHQL, VITE + VUE (typescript) and PostgreSQL v18 start-up project

Requirements:
1. Knowledge in C#, @vue/cli 5.0.8 and typescript
2. Node 24.13.0 and NPM 11.16.2
3. Knowledge in Vite, Vue, Typescript
4. Dotnet Core v10 CLI
5. POSTGRESQL V18 Server Docker Container
6. Bootstrap 5.3.7 and JQueryree-solid-svg-icons@7.0.0
7. Visual Studio Code
8. Install Google Authenticator in your Mobile Phone

Features :
1. Authentication and Authorization
   BryCrypt, JWT
2. Time Based One Time Password Authenticator
3. User Profile Picture upload/update
4. Product Listings and Pagination
5. User Account Activation via Email
6. Products Report - PDF
7. Annual Sales Chart

If you want to test, do the following:
1. Setup MSSQL 2025 DOCKER
2. Change Username and Password in appsettings.json, PostgreSQL connection settings
3. Run, dotnet ef migrations add InitialIdentitySchema and dotnet ef database
